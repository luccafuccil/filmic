# Filmic

A desktop video player application built with Electron and React for managing and watching your local movie library.

## Features

- Browse and play local video files
- Automatic thumbnail generation
- Continue watching functionality with progress tracking
- Movie metadata parsing from filenames
- Custom video player with keyboard shortcuts
- Modern, responsive UI

## Tech Stack

- Electron
- React
- Webpack
- FFmpeg for video processing

## Development

### Prerequisites

- Node.js
- npm

### Installation

```bash
npm install
```

### Running the Application

```bash
npm start
```

### Building

Build for Windows:

```bash
npm run dist:win
```

## Project Structure

- `electron/` - Main Electron process code
- `src/` - React application source
- `public/` - Static assets
- `dist/` - Built application output
